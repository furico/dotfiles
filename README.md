# 設定方法

macでのセットアップ方法。

## Homebrew

homebrewをインストールする。

- https://brew.sh/index_ja.html

## vim

clipboardオプションを有効にするため、brewでvimを入れる。

```
$ brew install vim --with-override-system-vi
```

## 設定ファイルをコピー

このリポジトリをcloneして `setup.sh` を実行する。

* `bash-it` がインストールされていることを前提としているため、先に `bash-it` をインストールしておくこと。
